  • Preguntas

  • ¿Con que trabajamos?

    • Trabajamos con VSC, que significa Visual Studio Code. Es una aplicacion donde se codea y en la misma se le agregan los estilos con CSS, SASS, SCSS, o linkeos externos para que quede a gusto de cada uno. Se puede hacer hasta el mas minimo detalle!

  • ¿De cuanto es el presupuesto de una pagina?

    • El presupuesto de una pagina puede variar segun varios factores, depende de cuantas modificaciones se le quiere hacer al proyecto, la mantencion, las actualizaciones, entre otras. Pero contactame que hablamos bien de que va tu proyecto para decirte con exactitud un presupuesto.

  • ¿En cuanto tiempo puedo tener mi pagina web?

    • Su pagina web debera estar lista en 1 o 2 semanas, siempre depende de las cosas que quieras meter o de lo grande que sea el proyecto pero eso seria un tiempo estimado.
